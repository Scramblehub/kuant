# kuant examples

End-to-end scripts showing how to compose kernels for real tasks.

Examples will accumulate as kernels land. Each script:

1. Imports only from kuant (no other quant deps)
2. Solves one specific task (backtest a signal, price an option surface, etc.)
3. Has a top-of-file comment explaining what it demonstrates
4. Runs on CPU by default (works without GPU); GPU is optional acceleration
